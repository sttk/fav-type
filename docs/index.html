@fav/type

Function set related to data types for all versions of nodejs and major browsers

"fav" is an abbreviation of "favorite" and also the acronym of "for all versions". This package is intended to support all Node.js versions and many browsers as possible. At least, this package supports NodeJS >= v0.10 and major Web browser: Chrome, Firefox, IE11, Edge, Vivaldi and Safari.

Install

For Node.js:

$ npm install --save @fav/type

Or (when npm < 2.7.0 which is not support scoped package)

$ npm install --save sttk/fav-type

For Web browsers:

<script src="fav.type.min.js"></script>

Usage

For Node.js:

const type = require('@fav/type');

const isString = type.isString
isString('abc') // => true

For Web browsers:

const isString = fav.type.isString
isString('abc') // => true
